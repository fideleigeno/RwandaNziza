# README

## Salary Report System

### Description

This project is a basic HTML and JavaScript program that calculates the net salary of 10 customers based on their gross salary.

### Features

* Enter the name of 10 customers.
* Enter the gross salary of each customer.
* Check if the salary entered is valid (greater than 0).
* Calculate tax using the following rates:

  * 1 – 60,000 Frw: **0%**
  * 60,001 – 100,000 Frw: **10%**
  * 100,001 – 200,000 Frw: **20%**
  * Above 200,000 Frw: **30%**
* Display the customer name, gross salary, tax rate, and net salary in a table.

### Technologies Used

* HTML
* JavaScript

### How to Run

1. Save the file with the `.html` extension.
2. Open the file in any web browser.
3. Enter the requested customer names and gross salaries.
4. View the salary report displayed in the table.
