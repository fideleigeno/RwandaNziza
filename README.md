# RwandaFix

## Overview

RwandaFix is a smart digital civic reporting platform that allows citizens to report infrastructure problems such as:

* Damaged roads
* Water leaks
* Broken street lights
* Electricity faults
* Flooding
* Waste management issues

The platform aims to improve communication between citizens and government authorities.
